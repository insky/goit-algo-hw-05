# goit-algo-hw-05

## Опис проекту
Цей проект містить реалізації алгоритмів пошуку підрядка: Кнута-Морріса-Пратта, Боєра-Мура та Рабіна-Карпа.
Кожен з цих алгоритмів має свої переваги та недоліки, і їх ефективність може змінюватися в залежності від конкретних умов використання. У цьому проекті ми порівняємо їх за часом виконання на різних текстах та шаблонах.

## Висновки
1. Алгоритм Боєра-Мура виявився найшвидшим серед трьох алгоритмів для більшості випадків, особливо при пошуку довгих підрядків.
2. Алгоритм Рабіна-Карпа виявився найповільнішим серед трьох, особливо при пошуку неіснуючих підрядків.
3. Всі алгоритми показали кращу продуктивність при пошуку існуючих підрядків порівняно з неіснуючими.
